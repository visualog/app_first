## 화면 전체 컨셉

- 모바일 기준의 **분석형 로또 대시보드 홈**.
- 구조: 헤더 → 회차 요약 → 1등 당첨번호 → 당첨 정보 카드 슬라이드.[^1][^2]

***

## 상단 헤더

- 페이지명: **홈**
- 우측: 사용자 아바타 아이콘
- 제거: 알림 아이콘, “안녕하세요 행운님” 인삿말
- 역할: 서비스 공통 헤더 + 현재 위치 표시.[^1]

***

## 메인 요약 영역

- 텍스트 형식:
    - `1024회 당첨 정보 · 2025-12-27 추첨`
- 기존 문구 제거:
    - “이번 주 로또 리포트”, “최신 당첨 정보를 한눈에 확인하세요.” 삭제
- 역할: 현재 회차와 추첨일을 한 줄로 명확하게 요약하는 정보 헤더.[^1]

## 1등 당첨번호 섹션

### 구조

- 섹션 타이틀: `1등 당첨번호`
- 번호 표현:
    - 예) `8  16  28  30  31  44  +  27`
    - 앞 6개: 1등 번호
    - `+` 뒤 1개: 보너스 번호.[^1]


### 볼(칩) 스타일

- 색상 전략
    - 기본 번호 6개: 실제 로또 볼 구간별 색을 기반으로 **채도를 낮춘 톤다운 컬러** 사용.[^3][^4]
    - 보너스 볼: 기본 번호와 다른 색 계열 또는 조금 더 진한 톤으로 구분.
- 형태
    - 원형 칩.
    - **그림자 제거**: 입체감·드롭섀도우 없이 플랫/세미플랫 스타일로 통일.[^5][^6][^7]
- 레이아웃
    - 위로는 회차 요약, 아래로는 카드 슬라이드와 시각적 무게가 충돌하지 않도록 적당한 여백.[^2][^1]

## 당첨 정보 카드 슬라이드

### 공통 UX

- 가로 슬라이드(스와이프) 카드 영역.
- 한 화면에 **카드 약 2.3~2.5장**이 보이도록 카드 폭 설정 → 슬라이드 가능성을 자연스럽게 인지.[^2]
- 하단에 페이지 인디케이터(도트) + “스와이프해서 더보기” 간단 안내.
- 모바일 기준: 탭 시 살짝 스케일/쉐도우/리플로 피드백, **호버 상태는 사용하지 않음**.[^2]


### 카드 공통 레이아웃

- 상단: 지표명(한글 타이틀)
    - 예: `1등 총 당첨금액`, `1등 당첨 인원` 등
    - 불필요한 영문 타이틀(First Prize Pool, Number of Winners 등)은 제거.[^1]
- 중앙: 핵심 숫자(큰 폰트 + 단위)
- 하단: 지난주 대비 정보
    - 예: `지난주 대비 +X억`, `지난주 대비 -Y명`
    - 작은 텍스트 + ▲▼ 아이콘으로 표시.
- 그래프:
    - 카드 하단에 **최근 5회(과거 4 + 최신 1)** 미니 바 그래프(스파크라인) 추가.
    - X축 라벨 없이 막대 5개만, 가장 오른쪽(현재 회차)만 강조 컬러 사용.[^2]


### 카드 목록

1. **1등 총 당첨금액**
    - 값: `299억`
    - 하단: `지난주 대비 +X억`
    - 미니 바 그래프: 최근 5회 총 당첨금 추이.[^1]
2. **1등 당첨 인원**
    - 값: `18명`
    - 하단: `지난주 대비 -Y명`
    - 미니 바 그래프: 최근 5회 당첨 인원 추이.[^1]
3. **1인당 1등 당첨금**
    - 값: `Z억`
    - 하단: `지난주 대비 +A억`
    - 미니 바 그래프: 최근 5회 1인당 당첨금 추이.[^2]
4. **1등 자동/반자동/수동**
    - 내용 예: `자동 10명 · 반자동 3명 · 수동 5명`
    - 필요 시 `지난주 대비 자동 +B명` 등 한 줄 텍스트.
    - 그래프 대신 텍스트/비율 중심 카드로 유지.[^2]
5. **1등 당첨 판매점 정보**
    - 내용 예: `오프라인 12곳 · 온라인 2곳` 또는 지역별 개수.
    - 필요 시 `지난주 대비 +C곳`.
    - 역시 그래프 없이 요약 정보 카드.[^2]
6. **1등 번호 합계**
    - 값: `157` (이번 회차 6개 번호 합계 예시)
    - 하단: `지난주 대비 +12`
    - 미니 바 그래프: 최근 5회 번호 합계 추이.[^2]

## 최종 화면 플로우 요약

1. **헤더**
    - 좌: 페이지명 **홈**
    - 우: 아바타
2. **메인 요약**
    - `1024회 당첨 정보 · 2025-12-27 추첨`
3. **1등 당첨번호 섹션**
    - 타이틀 `1등 당첨번호`
    - 톤다운된 실제 로또 볼 색 + 플랫 칩으로 `번호 6개 + + + 보너스 1개` 표시.[^4][^7][^3]
4. **당첨 정보 카드 슬라이드**
    - 한 화면에 2.3~2.5장 보이는 카드 슬라이드
    - 6개의 지표 카드(총 당첨금, 인원, 1인당 금액, 자동/반자동/수동, 판매점, 번호 합계)
    - 각 카드에 값 + 지난주 대비 + (필요 시) 최근 5회 스파크라인 그래프 제공.[^1][^2]
